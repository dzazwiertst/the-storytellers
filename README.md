# The Storytellers

Oficjalna strona zespołu The Storytellers.

## Praca lokalna

```bash
npm install
npm run dev
```

## Publikacja

Zmiany w gałęzi `main` są automatycznie budowane i publikowane na OVHcloud.
Workflow wymaga sekretów środowiska `production`:

- `OVH_FTP_SERVER`
- `OVH_FTP_USER`
- `OVH_FTP_PASSWORD`
